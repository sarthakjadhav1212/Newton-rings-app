# Newton-rings-app
# Newton's Rings ML Analyzer

This project analyzes Newton's Rings experimental data using Linear Regression.

## Features
- Manual / CSV / Excel input
- D² vs n graph
- ML vs Manual slope comparison
- Radius of curvature calculation
- Residual analysis

## Tech Used
- Python
- Streamlit
- Scikit-learn
- Matplotlib

## Run Locally
```bash
streamlit run app.py
